# Bit-Wallet
Safe and easy to use bitcoin wallet!

## Features

* HOT and COLD wallet modes to ensure private key safety
* Communication between hot and cold wallet via QR Codes
* SPV bitcoin p2p client
* Both android and iOS are supported
* Nice designed UI
